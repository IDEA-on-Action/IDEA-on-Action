## 🛠️ 캡처 방법 (Chrome 브라우저)

### 방법 1: Chrome DevTools (추천, 전체 페이지)

#### 단계별 가이드

**1. 브라우저 설정**
```
1. Chrome 브라우저 열기
2. 확대/축소: 100% (Ctrl+0)
3. 시크릿 모드 권장 (Ctrl+Shift+N)
4. 광고 차단기 OFF
```

**2. DevTools 열기**
```
1. F12 키 누르기 (또는 Ctrl+Shift+I)
2. DevTools 창 열림 확인
```

**3. 디바이스 툴바 활성화**
```
1. Ctrl+Shift+M 누르기
2. 상단 디바이스 선택: "Responsive"
3. 해상도 설정: 1920 x 1080
```

**4. 전체 페이지 스크린샷 캡처**
```
1. Ctrl+Shift+P 누르기 (Command Palette 열기)
2. "screenshot" 입력
3. "Capture full size screenshot" 선택
4. 자동으로 다운로드됨
```

**5. 파일 이름 변경**
```
다운로드 폴더에서 파일 찾기
→ 파일명 변경 (예: 01-services-list.png)
```

### 방법 2: Firefox (전체 페이지 스크린샷 내장)

```
1. Firefox 브라우저 열기
2. 페이지 로딩 완료 대기
3. 마우스 우클릭 → "스크린샷 찍기"
4. "전체 페이지 저장" 선택
5. 다운로드 후 파일명 변경
```

### 방법 3: 확장 프로그램 (GoFullPage)

```
1. Chrome 웹 스토어에서 "GoFullPage" 설치
   URL: https://chrome.google.com/webstore/detail/gofullpage/fdpohaocaechififmbbbbbknoalclacl
2. 확장 프로그램 아이콘 클릭
3. 자동으로 전체 페이지 캡처
4. PDF 또는 PNG 다운로드
5. 파일명 변경
```

## 📐 스크린샷 품질 요구사항

### 해상도
```
최소: 1920 x 1080
권장: 1920 x 1080 ~ 2560 x 1440
최대: 3840 x 2160 (4K, 선택 사항)
```

### 파일 형식
```
포맷: PNG (권장)
대체: JPG (JPEG Quality 90+)
금지: GIF, WebP, BMP
```

### 파일 크기
```
최대: 5MB/파일
권장: 500KB ~ 2MB
압축: TinyPNG (https://tinypng.com/)
```

```
브라우저 확대/축소: 100%
다크 모드: OFF (라이트 모드 권장)
광고 차단기: OFF
쿠키 배너: 닫기
Sticky 헤더: 표시됨 OK
```

## 🎯 자주 묻는 질문 (FAQ)

### Q1: 스크롤이 긴 페이지는 어떻게 캡처하나요?
**A**: Chrome DevTools의 "Capture full size screenshot" 사용하면 자동으로 스크롤 포함하여 전체 페이지를 캡처합니다.

### Q2: 모바일 화면도 필요한가요?
**A**: 아니오. 데스크톱 화면 (1920x1080)만 제출하면 됩니다. 반응형 디자인은 심사자가 브라우저 크기 조절로 확인합니다.

### Q3: 결제 페이지는 어떻게 캡처하나요?
**A**: 장바구니에 아이템을 추가한 후 `/checkout`으로 이동하여 캡처합니다. 실제 결제는 하지 않아도 됩니다.

### Q4: 구독 관리 페이지는 로그인이 필요한가요?
**A**: 예. 테스트 계정으로 로그인 후 `/profile/subscriptions`에서 캡처합니다. 구독이 없으면 "구독 없음" 메시지가 표시되어도 됩니다.

### Q5: 스크린샷에 개인정보가 포함되면 안 되나요?
**A**: 테스트 데이터 (예: test@example.com, 홍길동)를 사용하세요. 실제 개인정보는 모자이크 처리하거나 제거해야 합니다.

### Q6: 토스페이먼츠 결제 화면도 캡처해야 하나요?
**A**: 선택 사항입니다. 심사 시 토스페이먼츠 SDK가 정상 작동하는지 확인하려면 캡처해도 좋습니다 (결제 직전 화면).

### Q7: 다크 모드 스크린샷도 필요한가요?
**A**: 아니오. 라이트 모드 스크린샷만 제출하세요. 다크 모드는 선택 사항입니다.

## 📞 문제 해결

### 문제: 페이지가 잘려서 캡처됨
**해결**:
1. Chrome DevTools → "Capture **full size** screenshot" 확인
2. 또는 Firefox "전체 페이지 저장" 사용
3. 또는 GoFullPage 확장 프로그램 사용

### 문제: 파일 크기가 5MB 초과
**해결**:
1. TinyPNG (https://tinypng.com/)에서 압축
2. 또는 JPG 형식 (Quality 90)으로 변환
3. 또는 해상도 1920x1080으로 낮춤

### 문제: 이미지가 흐릿함
**해결**:
1. 브라우저 확대/축소 100% 확인
2. Chrome DevTools에서 해상도 1920x1080 설정
3. PNG 형식 사용 (JPG보다 선명)

### 문제: 법률 문서가 너무 길어서 캡처가 어려움
**해결**:
1. Chrome DevTools "Capture full size screenshot" 사용 (자동 스크롤)
2. 또는 여러 부분으로 나누어 캡처 후 합치기 (선택 사항)
3. PDF로 내보내기 (선택 사항)
