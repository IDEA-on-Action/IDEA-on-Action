## 📄 필수 서류 목록 (3개)

### 1. 사업자등록증 사본

**필요 정보**:
- 사업자등록번호: [사업자등록증에서 확인]
- 회사명: 생각과행동 (IdeaonAction)
- 대표자: 서민원
- 업태: 정보통신업 또는 소프트웨어 개발
- 종목: 웹/앱 개발, IT 컨설팅

**파일 형식**:
- PDF (권장)
- JPG/PNG (고해상도)
- 파일 크기: < 5MB

**준비 방법**:
1. **온라인 발급** (국세청 홈택스)
   - URL: https://www.hometax.go.kr
   - 로그인 → "민원증명" → "사업자등록증명" 발급
   - PDF 다운로드

2. **오프라인 발급**
   - 가까운 세무서 방문
   - 사업자등록증 사본 발급 (무료)
   - 스캔 또는 사진 촬영

3. **기존 사본 활용**
   - 기존에 발급받은 사업자등록증이 있다면 스캔
   - 해상도: 300 DPI 이상
   - 파일명: `business-registration-certificate.pdf`

**주의사항**:
- 사업자등록번호가 명확히 보여야 함
- 회사명, 대표자명이 웹사이트 정보와 일치해야 함
- 유효기간 없음 (최신 발급본 권장)

---

### 2. 대표자 신분증 사본

**필요 서류** (택1):
- 주민등록증 (앞면 + 뒷면)
- 운전면허증 (앞면 + 뒷면)
- 여권 (사진 페이지)

**파일 형식**:
- PDF (권장, 앞뒷면 통합)
- JPG/PNG (앞면 1개 + 뒷면 1개)
- 파일 크기: < 5MB

**준비 방법**:

#### 방법 1: 스캔 (권장)
```
1. 복합기 또는 스캐너 사용
2. 신분증 앞면 스캔 → 저장
3. 신분증 뒷면 스캔 → 저장
4. PDF 병합 (선택 사항)
   - 온라인 도구: https://www.ilovepdf.com/merge_pdf
   - 파일명: id-card-front-back.pdf
```

#### 방법 2: 사진 촬영
```
1. 스마트폰 카메라 사용
2. 밝은 곳에서 촬영 (그림자 없이)
3. 신분증 전체가 화면에 들어오도록
4. 초점 맞춰서 촬영 (흐릿하지 않게)
5. PC로 전송 후 파일명 변경
   - id-card-front.jpg
   - id-card-back.jpg
```

**개인정보 보호**:
- 주민등록번호 뒷자리 마스킹 (선택 사항)
  - 예: 900101-1******
- 토스페이먼츠는 본인 확인용으로만 사용
- 신청 완료 후 파일 삭제 권장

**주의사항**:
- 신분증 유효기간 확인 (만료되지 않았는지)
- 대표자 이름이 사업자등록증과 일치해야 함
- 얼굴 사진이 명확히 보여야 함

### 3. 회사 통장 사본

**필요 정보**:
- 은행명: [거래 은행 확인]
- 계좌번호: [통장 사본 또는 인터넷뱅킹]
- 예금주명: 생각과행동 (또는 서민원)

**파일 형식**:
- PDF (권장)
- JPG/PNG (통장 첫 페이지)
- 파일 크기: < 5MB

**준비 방법**:

#### 방법 1: 통장 사본 스캔
```
1. 통장 첫 페이지 (예금주, 계좌번호 표시)
2. 스캐너 또는 복합기 사용
3. PDF 저장
4. 파일명: bank-account-certificate.pdf
```

#### 방법 2: 인터넷뱅킹 인증서
```
1. 인터넷뱅킹 로그인
2. "계좌 정보 조회" 또는 "통장 사본"
3. PDF 다운로드 또는 화면 캡처
4. 예금주명, 계좌번호 확인
5. 파일명: bank-account-certificate.pdf
```

#### 방법 3: 은행 방문
```
1. 가까운 은행 지점 방문
2. 통장 사본 발급 요청 (무료 또는 소액)
3. 발급 후 스캔
```

**주의사항**:
- 예금주명이 사업자등록증과 일치해야 함
  - 법인: 회사명 (예: 생각과행동)
  - 개인사업자: 대표자명 (예: 서민원)
- 계좌번호가 명확히 보여야 함
- 잔액은 가려도 됨 (선택 사항)

## 🎯 자주 묻는 질문 (FAQ)

### Q1: 사업자등록증이 없으면 어떻게 하나요?
**A**: 토스페이먼츠는 **사업자 등록**이 필수입니다. 개인 명의로는 가맹점 신청이 불가능합니다. 개인사업자 또는 법인사업자 등록 후 신청하세요.

### Q2: 신분증 뒷자리를 가려도 되나요?
**A**: 예. 주민등록번호 뒷 7자리를 마스킹(***)해도 됩니다. 토스페이먼츠는 본인 확인용으로만 사용하며, 이름과 앞자리만으로도 충분합니다.

### Q3: 통장 사본 대신 거래내역서도 가능한가요?
**A**: 예. 예금주명과 계좌번호가 명확히 표시된 거래내역서(최근 3개월)를 제출해도 됩니다.

### Q4: 파일 크기가 5MB를 초과하면 어떻게 하나요?
**A**: 온라인 압축 도구를 사용하세요.
- PDF: https://www.ilovepdf.com/compress_pdf
- JPG: https://tinypng.com/

### Q5: 서류 유효기간이 있나요?
**A**:
- 사업자등록증: 유효기간 없음 (최신 발급본 권장)
- 신분증: 유효기간 내 (만료되지 않은 것)
- 통장 사본: 최근 3개월 이내 (권장)

### Q6: 외국인 대표자도 신청 가능한가요?
**A**: 예. 외국인등록증 또는 여권 사본을 제출하면 됩니다. 사업자등록증이 있어야 합니다.

### Q7: 법인과 개인사업자의 차이는?
**A**:
- **법인**: 법인등록번호, 법인인감증명서 추가 필요 (선택 사항)
- **개인사업자**: 사업자등록증, 신분증, 통장 사본만 필요

## 📞 문제 해결

### 문제: 사업자등록증을 분실했어요
**해결**:
1. 국세청 홈택스에서 재발급 (무료)
2. 또는 관할 세무서 방문 (즉시 발급)

### 문제: 신분증 사진이 흐릿해요
**해결**:
1. 밝은 곳에서 재촬영
2. 또는 스캐너 사용 (300 DPI 이상)
3. 스마트폰 카메라 초점 맞추기

### 문제: 통장이 없어요 (인터넷 전용 계좌)
**해결**:
1. 인터넷뱅킹에서 "통장 사본" 발급 (PDF)
2. 또는 계좌 정보 조회 화면 캡처 (예금주, 계좌번호 포함)

### 문제: 예금주명이 사업자등록증과 다릅니다
**해결**:
1. **법인**: 회사명으로 통장 개설 필요
2. **개인사업자**: 대표자명도 허용됨 (상호명 권장)
3. 불일치 시 은행에서 사업자 전용 계좌 개설 권장
