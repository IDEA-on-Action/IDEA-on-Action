# 쉬운말 국어사전 (EasyWord)

## 프로젝트 소개

**쉬운말 국어사전**은 다문화 가정 학생과 학습장애(난독증, 경계선 지능) 청소년을 위한 **맞춤형 단어장 서비스**입니다.

기존 사전은 어려운 한자어와 복잡한 설명으로 이루어져 있어, 어휘력이 부족한 학생들에게는 "사전을 찾아도 모르겠다"는 벽이 됩니다. 이 프로젝트는 **학생의 학년과 수준에 맞는 쉬운 설명**을 AI가 생성하고, **시각적 이미지**와 **일상적인 예문**을 함께 제공하여 단어 학습의 장벽을 낮추는 것을 목표로 합니다.

---

## 만들게 된 배경

학습장애 상담 현장에서 상담사들이 직접 학생 수준에 맞는 단어장을 만들어야 하는 어려움이 있었습니다. 매번 수작업으로 이미지를 찾고, 쉬운 설명을 작성하고, 예문을 만드는 과정이 반복되었습니다.

이 서비스는 **상담사와 특수교육 교사가 효율적으로 맞춤형 단어장을 제작**할 수 있도록 돕고, **학생들이 스스로 단어를 학습**할 수 있는 환경을 제공합니다.

## 핵심 기능

### 1. 이미지 선택
단어를 입력하면 **구글 검색 이미지 3개 + AI 생성 이미지 1개**를 2x2 그리드로 제시합니다. 시각적 학습이 효과적인 학생들을 위해 직관적인 이미지를 선택할 수 있습니다.

### 2. 사전적 의미 + 난이도 표시
네이버 사전에서 정확한 뜻을 가져오고, AI가 **단어 난이도(1~5단계)**를 분석하여 표시합니다. 학생이 어떤 단어가 어려운지 한눈에 파악할 수 있습니다.

### 3. 학년별 쉬운 뜻 설명
**초등 저학년, 초등 중학년, 초등 고학년, 중등, 고등** 5단계로 학년을 선택하면, AI가 해당 수준에 맞는 쉬운 우리말로 뜻을 설명합니다.

예시) "민주주의"
- 초등 저학년: "나라의 중요한 일을 모든 사람이 함께 정하는 것이에요"
- 고등: "국민이 주권을 가지고 직접 또는 대표를 통해 국가 의사결정에 참여하는 정치 체제"

### 4. 일상 예문
각 뜻에 맞는 **일상생활에서 자주 쓰이는 쉬운 예문**을 AI가 생성합니다. 학생이 실제로 사용할 수 있는 맥락을 제공합니다.

### 5. 출처 예문
**교과서, 문학작품, 뉴스 기사**에서 해당 단어가 사용된 예문을 제시하고 출처를 명시합니다. 학년별로 적절한 출처(초등 교과서, 동화, 어린이 신문 등)를 선별합니다.

### 6. 다국어 지원
베트남어, 중국어 등 **다문화 가정 학생의 모국어**로 번역을 제공합니다.

### 7. 음성 지원 (TTS)
모든 단어와 설명에 **듣기 버튼**이 있어 발음을 확인할 수 있습니다.

## 기술 스택

| 영역 | 기술 |
|------|------|
| **프론트엔드** | Next.js 14, TypeScript, Tailwind CSS, shadcn/ui |
| **백엔드** | Cloudflare Workers, Hono, Drizzle ORM |
| **데이터베이스** | Cloudflare D1 (SQLite) |
| **AI 서비스** | Claude API (쉬운 뜻, 예문), Gemini API (이미지 생성) |
| **외부 API** | Google Custom Search (이미지), Naver 사전/Papago (번역) |
| **모바일** | Capacitor (Android/iOS 앱), PWA |
