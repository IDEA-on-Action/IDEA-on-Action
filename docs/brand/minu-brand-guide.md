# 🌱 **1. Minu 브랜드 스토리 (Brand Story)**

우리는 거대한 목표를 생각하지만,
실제로 움직이게 만드는 건 언제나 작은 시작이다.

Minu는 그런 작은 시작의 이름이다.
하나의 단서, 작은 아이디어,
문제의 작은 조각을 붙잡아
자연스럽게 다음 단계로 이어지게 만든다.

사업기회가 발견되고(Find),
생각이 틀을 잡고(Frame),
실행이 쌓이고(Build),
운영으로 이어지고(Keep),
또다시 다음 시작을 만든다.

Minu는
프로젝트라는 긴 여정의 모든 순간에
과도한 힘을 주지 않고,
가볍고 자연스럽게
생각을 행동으로 이어지게 돕는다.

작게 시작해도 괜찮다.
언제든 다시 시작할 수 있으니까.
Minu는 모든 시작을 가볍게 만들기 위해 존재한다.

# 🔍 **4. 서비스 상세 설명 (각 제품별)**

## **Minu Find – 사업기회 탐색 서비스**

작은 신호에서 사업기회를 발견합니다.
시장·기술·정책·경쟁사 정보를 자동으로 모아
“할 만한 기회”만 깔끔하게 보여줍니다.
프로젝트의 가장 앞단을 가볍게 열어주는 도구입니다.

📌 주요 기능

* 산업·기술 시그널 수집
* 경쟁사 모니터링
* 사업기회 자동 스코어링
* 원페이저 기회 분석 생성

---

## **Minu Frame – 문제 정의 & RFP 설계 서비스**

올바른 문제 정의는 좋은 프로젝트의 절반입니다.
Minu Frame은 생각을 구조화하고, 요구사항을 정리해
기획 초기 단계의 혼탁함을 단숨에 정리합니다.

📌 주요 기능

* 문제 정의 Wizard
* 요구사항 후보 자동 생성
* 기능/비기능 자동 구분
* RFP 1.0 자동 생성

---

## **Minu Build – 개발 & 프로젝트 진행 서비스**

프로젝트는 복잡하기 쉽지만,
진행 상황을 이해하는 일까지 복잡할 필요는 없습니다.
Minu Build는 일정·이슈·진척을 부드럽게 요약해
PM/PL이 관리 대신 본질에 집중할 수 있게 만듭니다.

📌 주요 기능

* 일정/이슈/리스크 요약
* Sprint/모듈 진척 자동 분석
* 주간보고 자동 생성
* Git/Jira 경량 연동

---

## **Minu Keep – 운영/유지보수 서비스**

운영은 “묵직하게”가 아니라 “가볍고 지속 가능한 방식”이어야 합니다.
Minu Keep은 버그, 업데이트, 리소스, 운영 기록을
단정하면서도 가벼운 방식으로 정리합니다.

📌 주요 기능

* 버그/작업 자동 분류
* 운영 보고서 생성
* 배포/업데이트 관리
* 운영 리소스 추적

# 🪄 **6. 시리즈 아이콘 시스템**

각 제품(Minu Find, Frame, Build, Keep)을
**씨앗의 성장 과정**으로 상징화하는 미니멀 아이콘 시스템으로 설계:

* **Find:** 땅 속 작은 씨앗 아이콘(•)
* **Frame:** 씨앗에서 싹이 트는 형태(⌒)
* **Build:** 잎이 두 장 열린 작은 식물(🌱)
* **Keep:** 식물이 완전히 자리 잡은 안정된 모양(🌿)

형태적 차이는 *크기, 잎의 개수, 구조*만 변형하면 되므로
일관성과 브랜드 통일성이 매우 강해짐.

## **Minu**

**Minu는 Find → Frame → Build → Keep**
프로젝트 전 과정을 가볍고 자연스럽게 이어주는 4단계 SaaS입니다.

---

### 🌱 **Why Minu? (Minu가 왜 필요한가)**

* 프로젝트는 언제나 **작은 단서에서** 시작된다
* 그러나 기획부터 운영까지 **각 단계는 단절적으로 분리**되어 있다
* 대부분의 SaaS는 All-in-One으로 과하게 무겁다
* Minu는 **단계별로 가볍게**, **필요할 때만** 사용하도록 설계되었다

---

### 🌿 **What Minu does (무엇을 해결하는가)**

**1) Minu Find – 사업기회 발견**
시장·기술 신호를 자동 수집하고 ‘할 만한’ 기회를 정리합니다.

**2) Minu Frame – 문제정의 & RFP**
생각을 구조화하고, 요구사항을 빠르게 틀 잡아줍니다.

**3) Minu Build – 프로젝트 실행**
일정·이슈·진척을 자동으로 요약하여 실행 흐름을 잡습니다.

**4) Minu Keep – 운영/유지관리**
운영 티켓·리소스·리포트를 단정하게 관리할 수 있습니다.

---

### 🌼 **Minu Philosophy (철학)**

> “작은 시작이 행동을 만든다.”
> Minu는 생각과 행동 사이의 간극을 부드럽게 연결합니다.

---

### ▶ **Minu의 핵심 가치**

* 부담 없이 시작하는 **경량 SaaS**
* 필요할 때만 쓰는 **단계별 구독 모델**
* 생각 → 정의 → 실행 → 운영의 **일관된 흐름** 구축
* 브랜드 철학(IDEA on Action)과 자연스럽게 이어지는 서비스
